This is a Next.js + Tailwind site ready to deploy on Vercel.

## Quick Start
1. Install: `npm install`
2. Run locally: `npm run dev` (visit http://localhost:3000)
3. Deploy to Vercel: Import this repo at https://vercel.com/new

## Edit Text/Images
- Open `app/page.jsx` and change the `defaultData` object.
- In production, small text edits can be done via GitHub's web editor; Vercel will auto-redeploy.
